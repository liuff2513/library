# 中文简历解析
> Edited by feifei.liu 20200910

## 一、简历通用类别和详细定义

| 通用类别     | 详细信息                                   |
| ------------ | ------------------------------------------ |
| 个人基本信息 | 姓名、性别、生日、住址、手机号码、邮箱     |
| 求职意向     | 工作性质、期望工作地点、期望职位、期望薪资 |
| 自我评价     | 专业技能、性格特点                         |
| 教育经历     | 时间、学校、专业、学历                     |
| 工作经历     | 工作时间、工作名称、职位名称、工作内容     |
| 其他         | 其他详细信息                               |
